# Maximum de Trois Nombres en C

## Description
Trouve le maximum parmi 3 réels via une fonction dédiée
et identifie sa position.

## Objectif pédagogique
- Fonction avec valeur de retour
- Algorithme de recherche de maximum

## Compilation

```bash
gcc -o maximum_trois main.c
./maximum_trois
```
