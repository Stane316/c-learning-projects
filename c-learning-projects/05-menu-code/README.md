# Menu avec Code de Sélection en C

## Description
Programme qui affiche un menu à 3 options, lit le choix de
l'utilisateur et exécute l'action correspondante via `switch`.

## Objectif pédagogique
- Instruction `switch/case` avec `default`
- Validation de plage (1-3)
- Menu interactif avec boucle de relance

## Compilation

```bash
gcc -o menu_code main.c
./menu_code
```
