# Maximum de N Nombres en C

## Description
Lit N nombres et trouve le maximum avec sa position.

## Objectif pédagogique
- Boucle `for` avec saisie progressive
- Mise à jour du maximum en cours de lecture
- `#define MAX_NOMBRES` pour limiter la saisie

## Compilation

```bash
gcc -o maximum_n main.c
./maximum_n
```
