# Minimum avec Sentinelle en C

## Description
Lit des réels jusqu'à la sentinelle `0`, puis affiche le minimum.

## Objectif pédagogique
- Valeur sentinelle pour terminer une saisie
- Boucle `while` indéfinie
- Initialisation du minimum au premier élément

## Compilation

```bash
gcc -o minimum_sentinelle main.c -lm
./minimum_sentinelle
```
