# Minimum d'un Tableau en C

## Description
Saisit 10 entiers dans un tableau et trouve le minimum
avec son indice et sa position (base 1).

## Objectif pédagogique
- Tableaux en C (`int tab[TAILLE]`)
- Parcours avec `for`
- Distinction indice (base 0) / position (base 1)

## Compilation

```bash
gcc -o minimum_tableau main.c
./minimum_tableau
```
