# Tri d'Étudiants par Moyenne en C

## Description
Saisit N étudiants (nom, prénom, moyenne), les trie par moyenne
décroissante (tri à bulles) et affiche le classement.

## Objectif pédagogique
- `struct` pour regrouper des données
- Tri à bulles décroissant
- Tableau de structs

## Compilation

```bash
gcc -o tri_etudiants main.c
./tri_etudiants
```
