# Commande de Produits en C

## Description
Sélectionne un type de produit, saisit une quantité bornée
au stock disponible et calcule le total.

## Objectif pédagogique
- `enum` pour les types de produits
- Validation de quantité avec borne dynamique (stock)
- `const char*` retourné par une fonction switch

## Compilation

```bash
gcc -o commande_produit main.c
./commande_produit
```
