# Maximum d'un Tableau en C

## Description
Saisit 10 entiers et trouve le maximum avec son indice.
Le maximum est mis en valeur entre crochets à l'affichage.

## Objectif pédagogique
- Tableaux et parcours linéaire
- Affichage conditionnel (crochets sur le max)

## Compilation

```bash
gcc -o maximum_tableau main.c
./maximum_tableau
```
