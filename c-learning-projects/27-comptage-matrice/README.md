# Comptage de Valeurs dans une Matrice en C

## Description
Compte les valeurs positives, négatives et nulles
d'une matrice et affiche les pourcentages.

## Objectif pédagogique
- Compteurs multiples sur une matrice
- Pourcentages calculés à la fin
- Comparaison à zéro avec epsilon pour les flottants

## Compilation

```bash
gcc -o comptage_matrice main.c -lm
./comptage_matrice
```
