# Maximum d'une Matrice en C

## Description
Saisit une matrice 3×3 et trouve la valeur maximale
avec sa position (ligne, colonne).

## Objectif pédagogique
- Tableaux 2D en C
- Boucles imbriquées
- Affichage conditionnel sur la cellule maximale

## Compilation

```bash
gcc -o maximum_matrice main.c
./maximum_matrice
```
