# Valeurs Impaires d'un Tableau en C

## Description
Saisit 10 entiers et affiche uniquement les valeurs impaires
avec leur ratio.

## Objectif pédagogique
- Filtrage dans un tableau
- Compteur conditionnel
- Ratio impairs/total

## Compilation

```bash
gcc -o valeurs_impaires main.c
./valeurs_impaires
```
