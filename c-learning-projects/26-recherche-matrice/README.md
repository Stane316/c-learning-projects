# Recherche dans une Matrice en C

## Description
Recherche une valeur dans une matrice de taille variable
et affiche toutes ses occurrences.

## Objectif pédagogique
- Matrice à taille dynamique (jusqu'à 5×5)
- Comparaison de flottants avec epsilon
- Comptage des occurrences

## Compilation

```bash
gcc -o recherche_matrice main.c -lm
./recherche_matrice
```
