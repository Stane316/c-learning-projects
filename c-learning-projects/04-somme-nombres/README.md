# Somme de Deux Nombres Réels en C

## Description
Programme qui lit deux réels, calcule leur somme et affiche
le signe du résultat (positif, négatif ou nul).

## Objectif pédagogique
- Saisie et traitement de nombres réels
- Conditions sur le signe d'un résultat
- Comparaison sécurisée à zéro avec epsilon

## Exemple

```
Nombre 1 : 3.50
Nombre 2 : -5.20
3.50 + (-5.20) = -1.70
Resultat : NEGATIF
```

## Compilation

```bash
gcc -o somme_nombres main.c -lm
./somme_nombres
```
